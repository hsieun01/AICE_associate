# 자동차 가격 예측
## 자동차 가격 데이터를 읽어들여 데이터 분석 및 전처리 후 머신러닝과 딥러닝으로 자동차 거래 가격 예측
### 원본 데이터 변수
-year: 차량 제조 연도
-make: 브랜드 원산지
-model: 모델명
-trim: 트림(세부 등급)
-transmission: 변속기 유형
-vin: 차량 식별 번호
-state: 거래가 이루어진 주
-condition: 차량 상태 점수
-odometer: 주행 거리
-interior: 차량 내부 색상
-seller: 판매자
-mmr: MMR(해당 모델의 시장 평균가)
-sellingprice: 거래 가격
-saledate: 거래 일자
-body_type: 차체 유형
-color_group: 색상 계열
